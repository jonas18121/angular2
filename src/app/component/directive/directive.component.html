Les Directives

  • Les directives structurelles : Elles ont pour but de modifier le DOM en ajoutant, enlevant ou replaçant un élément du DOM.
  • Les directives attributs : Elles ont pour but de modifier l'apparence ou le comportement d'un élément.
  • Tous les attributs HTML existent sous la forme d'une directive Angular
